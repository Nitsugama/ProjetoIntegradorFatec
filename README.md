Para rodar o projeto:
Rode o script SETUP_MYSQL.sql no MySql para criar o banco de dados.

No projeto, na pasta backend, instale as dependencias (npm install) e rode o servidor (npm start)
Na raiz do projeto, instale as dependencias (npm install) e rode o frontend (npm run dev)
